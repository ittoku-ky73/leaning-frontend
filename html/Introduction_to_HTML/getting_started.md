# HTMLを始めよう

- 行を独立させたい場合、`p`要素で囲んで段落指定すると良い

## HTML要素の分解

![grumpy-cat-small](https://mdn.mozillademos.org/files/9347/grumpy-cat-small.png)

- 行をイタリック体（斜体）にしたい場合、`em`要素で囲むと良い

## 要素のネスト

- 要素の中に要素を入れることをネスト（入れ子）という。下記ではveryの部分が強調される

```html
<p>My cat is <strong>very</strong> grumpy.</p>
```

## ブロック要素とインライン要素

- ブロック要素
  - ウェブページ上で視認できるブロックを形成します 。
  - それらはそれより前にあるいかなるコンテンツに対しても新たな行におけるコンテンツとして表示され、そのブロックの後に来るいかなるコンテンツもまた新たな行で表示されることになります
  - 段落・リスト・ナビゲーションメニュー・フッターなどを表すことに使用される傾向がある
- インライン要素
  - ブロックレベル要素の中に包含され、なおかつ、段落全体やコンテンツのグループではなく、ドキュメントの内容の小さな部分だけを囲む要素
  - また、新たな行として表示されない。
  - a, em, strongタグといった強調要素のように、テキスト段落の中で表示される

```html
<!-- 例 -->
<em>first</em><em>second</em><em>third</em>

<p>fourth</p><p>fifth</p><p>sixth</p>
Copy to Clipboard
```

## 空要素

- 全ての要素が開始タグ・コンテンツ・終了タグのパターンに従っているわけではない

```html
<img src="https://raw.githubusercontent.com/mdn/beginner-html-site/gh-pages/images/firefox-icon.png">
```

## 属性

![grumpy-cat-attribute-small](https://mdn.mozillademos.org/files/9345/grumpy-cat-attribute-small.png)

- 属性をつける条件
  1. 属性名と要素名の間に 1 つの半角スペース (その要素内にすでに 1 つ以上の属性が設定されている場合は、併せて各属性名の間)
  2. 属性名とそれに続く等号 (=)
  3. 属性値。始端から終端までをクォーテーションマーク (引用符) で囲む

- `href`: この属性にはリンクが参照するウェブ上のアドレスを指定します
- `title`: この属性にはリンクについての追加情報（リンク先のページが何であるかなど）を記載します
- `target`: リンク先の内容を表示する場所 (ブラウジングコンテキスト) を指定します
  - `target="_blank"` と指定すると、リンク先の内容を新しいタブに表示される

### 真偽値属性

```html
<input type="text" disabled="disabled">

<!-- 略式 -->
<input type="text" disabled>
```

## HTML文書の構成

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>My test page</title>
  </head>
  <body>
    <p>This is my page</p>
  </body>
</html>
```

1. `<!DOCTYPE html>`: いわゆる "doctype" というものです。 HTML の黎明期(1991～1992年あたり)には、 "doctype" というものは HTML の文法セットへのリンクとしての役割を持っていました
2. `<html></html>`: ページ全体を指す
3. `<head></head>`:HTML ページに入れておきたいが見せたくないすべてのものを入れておくコンテナーの働きをする
   - 例えば検索結果に出したいページのキーワードや説明や、ページのスタイルを指定するための CSS や、文字エンコーディングの定義など
4. `meta charset="utf-8"`: この文書が UTF-8 という文字コードを使用しているということをブラウザーに伝えるためのものです。UTF-8 は世界中の自然言語の大半をカバーしている文字コードです。重要なこととしてあらゆるテキストコンテンツを扱うことができます。文字コードとして UTF-8 を指定しない手
5. `<title></title>`: ページのタイトルを指定するもので、ページが読み込まれたブラウザーのタブに表示されます
6. `<body></body>`: この中にユーザーがページを訪問した時に表示したいコンテンツ (例えば、テキスト、画像、ビデオ、ゲーム、オーディオトラック等) を記述します。

### HTML内の空白

- 大量の半角スペースを挿入しても、１つの半角スペースとして処理される
- 改行したければ`br`要素を使う

## HTMLの特殊文字

- HTMLでは、`<`、`>`、`"`、`'`、`&`の５つの文字は特殊文字と呼ばれていて、HTMLの文法自身の一部。
- 表示するには下記のように記述する

| 実際の文字 | 等価な文字参照 |
| :--------- | :------------- |
| <          | \&lt;          |
| >          | \&gt;          |
| "          | \&quot;        |
| '          | \&apos;        |
| &          | \&amp;         |
